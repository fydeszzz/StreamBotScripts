===============================================================================
 Name: 		Call Dragon for Streamlabs Bot
 Description: When users used command at specified times, dragon will appear in chat and give users prize.
 Version: 	1.0.0
 Creator: 	Tristia
 Website:	https://github.com/fydeszzz/StreamBotScripts
 Recently Release Date: 2020/12/4
===============================================================================
 - Enable/Disable using the command when the stream is offline
   是否限定實況中才能使用
 - Enable/Disable cooldown / user cooldown
   自訂是否使用指令/使用者冷卻時間
 - Enable/Disable cooldown / user cooldown / restart messages
   自訂冷卻時間與重置提示訊息
 - Customisable command name
   自訂指令名稱
 - Customisable permission for using the command
   自訂使用者權限(所有人、訂閱者等)
 - Customisable goal for calling dragon
   自訂達成獎勵目標次數
 - Customisable winning prize
   自訂獎勵代幣數量
 - Customisable response messages
   自訂回應訊息
===============================================================================
 Usable Variable 可使用參數定義
 
 $username : username 使用者名稱
 $cd : cooldown time of command 冷卻時間(指令)
 $command : command 自訂指令名稱
 $rewardlist : winners 獎勵名單(含個別使用者獲得獎勵數)
 $progress : progress of customisable goal 達成目標進度
 ※根據使用者設定的目標指令次數
